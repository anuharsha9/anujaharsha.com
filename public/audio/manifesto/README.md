# Manifesto Audio Drop Zone

To power the 60-second interactive presentation, you just need a single audio track.

## Instructions
1. Record your 60-second voiceover reading the script.
2. Export the file as an `.mp3`, `.wav`, or `.m4a` file.
3. Rename the file to exactly:
   **`manifesto-full.mp3`** (or `.wav` / `.m4a`)
4. Drop it into this folder (`/public/audio/manifesto/`).

The Next.js application is already wired up to look for `/audio/manifesto/manifesto-full.mp3` and will automatically sync its playback and seeking with the visual timeline scrubber.

## Tip for LinkedIn
If you plan to use a screen recording of this for LinkedIn, having everything synced perfectly to one master track will make capturing the final video flawless.
